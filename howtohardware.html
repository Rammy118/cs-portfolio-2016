How To

You start html by putting in this code: 

                <!doctype html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
The html start tag is always at the very beginning of the document's coding. Then the head tag is what the characteristics of the document are like the background color, font color, etc.
To give the website a background color, you put in this code: 
                <style>
        body {
            background-color:any color you want;
        }
    </style>
In between the style tags is the colors of what you want to color like font, background (like shown in the code), etc. After doing that you put the end tag for head: 
                </head>

For the links I added in a navbar using the ul tag with the id of navbar. The links have the li tag with the a tag as well with the href id. For the paragraphs you need to have the h tags with h1 being the title of the page and h2 being the title of the paragraph. The p tag is the paragraph. The start p tag has the id of the title of the first header (the h1). For images you need to have the img tag with the src id and the image name and alt id for the paragraph (the h2 header) you want the image to go under. To have the images show you have to go to File and Upload Local Files and select the files you want. You should always have the end tag for html at the very end of the document. You also use the body element for what you see on the page, the h tags for headers (which go from h1 to h6), the p element for paragraphs, and img element for images.
