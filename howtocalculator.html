How To

To add in the text boxes for numbers one and two, you add in this html code: 

                    Number 1 <input id="number1"> <br>
                    Number 2 <input id="number2"> <br> <br>
The input id is what you are supposed to put in the text box next to it is. The break tags are so that they can be on two different lines. For the dropdown menu, you have to add in this code: 
                    Operation <br>
        <select id="operation">
            <option value="Add">+</option>
            <option value="Sub">-</option>
            <option value="Mul">*</option>
            <option value="Div">/</option>
            <option value="GT"> Greater Than</option>
            <option value="EQ"> Equal</option>
        </select> <br> <br>
The option values are what the options of the dropdown menu are supposed to do to the number (like add, subtract, multiply, or divide the two numbers you put or determine if the number in the first text input is greater than the number in the second input or to determine if they are both equal. If you put two different numbers and selected the greater than option, the result would say "true" if the number in the first input is larger, or "false" if the number in the second input is larger. If you put in two of the same number and selected the equal option and clicked calculate, it would say "true", but if you did just that, except the two numbers are different, then it will say "false") To add the button, you put in this code: 
                     <button id="calculate">Calculate</button><br><br>
The button id is what gives the button its function. To make the result showing up after clicking the calculate button possible, you start by adding in this code: 
                <div id="result"></div>
This div function is what makes the result show up after clicking the calculate button. To link the html to the javascript, you add in this code: 
                <script src="calculator.js"></script>
The script tag is what links the html to the javascript so that the codes in the javascript can be used on the document.
