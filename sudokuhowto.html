// Store the new game button in a variable

This variable is the variable I used to make the new game button that when pushed, it will generate numbers on certain cells in the sudoku game. This is what will start the game, which the function is given to it by the get element by id with the function, "startnewgame".

// Store all the sudoku cells in an array variable

var cells = document.getElementsByClassName("cell");

This is an array that makes the small sections, rows, and columns of the sudoku. They are in boxes and small sections are a tiny bit apart from each other. They also have dropdown menus containing the numbers you put into the cell for every cell. They all have the class name "cell".
